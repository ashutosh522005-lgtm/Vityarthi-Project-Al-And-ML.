# Smart Travel Planner 🌍

## Overview

Smart Travel Planner is a Python-based desktop application that suggests travel destinations based on the user's **mood, budget, and purpose**.
The project is built using **Python Tkinter GUI** and **AI rule-based decision logic** without using any external API, making it fully offline and lightweight.

This project demonstrates basic **Artificial Intelligence concepts such as Knowledge Representation, Intelligent Agents, and Decision Making Systems** in a simple and interactive way.

---

## Features

* Dark themed modern UI
* Suggests travel destinations based on:

  * Mood
  * Budget
  * Purpose
* Expanded travel database
* Always provides at least one suggestion
* Offline system (No API required)
* Easy-to-use dropdown interface
* Clear and Suggest buttons
* Multiple travel suggestions

## Installation

### Step 1: Install Python

Download and install Python from:

https://www.python.org/downloads/

Make sure to check **Add Python to PATH**.

---

### Step 2: Install Tkinter

Tkinter comes pre-installed with Python.

To verify:

```
python -m tkinter
```

If a window opens, Tkinter is working.

## How It Works

1. User selects Mood
2. User selects Budget
3. User selects Purpose
4. Click Suggest Place
5. System checks travel database
6. AI rule-based logic finds matching places
7. Suggested destinations are displayed

If no exact match is found, similar places are suggested.
